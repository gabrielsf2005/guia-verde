Este projeto de extensão universitária tem como objetivo fornecer uma ferramenta prática, visual e acessível para pequenos produtores rurais — com foco inicial no **Distrito Federal (DF)**. O sistema auxilia na identificação e no manejo de patologias que afetam as principais culturas da região.

## 🚀 O Projeto

Diferente de sistemas mais robustos e complexos, este projeto utiliza a arquitetura de **Microsite Estático (Master-Detail)**. Isso garante que a ferramenta seja leve, rápida e funcional, mesmo em áreas rurais com conectividade limitada.

* **Página Principal:** Painel intuitivo com cards de acesso rápido para cada cultura.
* **Páginas de Detalhes:** Guias específicos para cada praga ou doença, contendo fotos e recomendações técnicas.
